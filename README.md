# 💼 Projeto Site de Apresentação

Projeto de portfólio desenvolvido com **HTML e CSS**, com foco na criação de uma interface moderna, organizada e responsiva.

## 🚀 Sobre o projeto

Este projeto foi criado como parte do meu processo de aprendizado em desenvolvimento front-end.  
O objetivo é servir como base sólida de aplicação de conhecimentos.

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- Flexbox  

## 🎨 Funcionalidades

- Layout responsivo  
- Estrutura semântica  
- Navegação simples  
- Estilização moderna com foco em legibilidade
